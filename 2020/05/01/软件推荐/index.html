Windows10软件推荐
发表于 2020-05-01| 更新于 2020-06-01|软件推荐
字数总计:251|阅读时长: 1 分钟
| 阅读量:|评论数:

使用技巧

  • ScreenToGif
    • 获取屏幕动态gif图（可修改删除）
  • Listary
    • 这使得查找您的文件和启动应用程序的速度非常快
  • Snipaste
    • 截图工具，可屏幕固定屏幕截图，可以获取选中区域颜色值
  • 7zip
    • 压缩软件轻巧免费

视频播放

  • PotPlayer
    • 轻巧 功能强大

文本编辑

  • Sublime Text3
  • Visual Studio Code
  • notepad++

系统安全

  • 火绒安全软件
    • 界面简洁，足够使用

记笔记

  • 有道云笔记
    • 手机电脑端同步笔记，界面简洁

git管理

  • GitKraken
    • GitKraken是最好的最好的Git客户端，是一款Git托管服务集成，能方便您更容易使用远程存储。

md文档编辑

  • Typora
    • 界面很舒服，很享受写文档

命令行工具

  • cmder
    • 一款Windows环境下非常简洁美观易用的cmd替代者，它支持了大部分的Linux命令。支持ssh连接linux，使用起来非常方便。
